# 🚀 SMETA.MD Bot — Деплой на Railway

## Что это
Telegram бот-сметчик для электромонтажных работ (Молдова, MDL).
- Принимает текст, PDF, фото проектов
- Составляет сметы: черновой + чистовой этап
- Синхронизирует с Google Sheets CRM

---

## Шаг 1 — GitHub репозиторий

1. Зайди на **github.com** → New repository
2. Назови: `smetamd-bot`
3. Загрузи 3 файла:
   - `bot.py`
   - `requirements.txt`
   - `README.md`

---

## Шаг 2 — Railway деплой

1. Зайди на **railway.app**
2. **New Project** → **Deploy from GitHub repo**
3. Выбери `smetamd-bot`
4. Railway сам определит Python проект

---

## Шаг 3 — Переменные окружения (ВАЖНО!)

В Railway → твой проект → **Variables** → добавь:

| Переменная | Значение |
|---|---|
| `ANTHROPIC_API_KEY` | `sk-ant-api03-...` (твой ключ) |
| `TELEGRAM_TOKEN` | `8776654437:AAGXhvSg...` |
| `APPS_SCRIPT_URL` | `https://script.google.com/macros/s/AKfycbw7...` |

⚠️ **Никогда не храни ключи в коде — только в переменных Railway!**

---

## Шаг 4 — Start команда

В Railway → Settings → **Start Command**:
```
python bot.py
```

---

## Шаг 5 — Проверка

1. Открой Telegram → найди своего бота
2. Напиши `/start`
3. Должно появиться приветствие с кнопками

---

## Команды бота

| Команда | Действие |
|---|---|
| `/start` | Начать новую смету |
| `/new` | Сбросить историю |
| `/help` | Справка |

## Как пользоваться

**Вариант 1 — Текстом:**
```
ID0199, квартира Чеканы, 3 комнаты, 78м², панельный дом
```

**Вариант 2 — PDF проект:**
Просто отправь PDF в чат → бот прочитает план и составит смету

**Вариант 3 — Фото:**
Сфотографируй план → отправь → бот разберёт

## Troubleshooting

**Бот не отвечает:**
- Проверь TELEGRAM_TOKEN в Variables
- Проверь Railway Logs на ошибки

**Ошибка API:**
- Проверь ANTHROPIC_API_KEY
- Проверь баланс на console.anthropic.com

**CRM не синхронизируется:**
- Проверь APPS_SCRIPT_URL
- Убедись что Apps Script задеплоен как веб-приложение
